The Challenge

The organization needed a modern website that clearly communicated its mission, shared inspiring stories, and provided a seamless user experience for donors, volunteers, and visitors. It was essential for the site to be easily maintainable, accessible, and optimized for performance across all devices.

The Solution

I built the website using React and TypeScript, applying a component-based architecture for scalability and efficient content management. Tailwind CSS and custom CSS were used to deliver a visually appealing, responsive, and consistent interface. The design includes carousel sections for storytelling, SVG icons for intuitive navigation, and subtle hover animations to enhance interactivity.

The backend was powered by Node.js with MSSQL and phpMyAdmin for database management. All images and static files were optimized and served through Cloudinary CDN for faster load times. The project was hosted on Namecheap with a custom domain, and version control was maintained using Git.
